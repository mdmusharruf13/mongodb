## MongoDB Practice

### Start Server on Windows

```
mongosh
```

### create or use Database

```
use mydb;
```

### list all Databases

```
show dbs;
```

### list collections

```
show collections;
```

### Create Collection in Database

```
db.createCollection(collectionName);
```
